===== S3 =====

- create a bucket -> aws s3 mb s3://bucket-name
- list bucket -> aws s3 ls s3://bucket-name
- upload file -> aws s3 cp filename.ext s3://bucket-name --acl public-read


===== LAMBDA =====

- update function code -> aws lambda update-function-code --zip-file=fileb://filename.zip --function-name myfunction
- update function config -> aws lambda update-function-configuration --function-name myfunction --handler newname.handler
- update function version -> aws lambda update-function-code --zip-file=fileb://filename.zip --function-name myfunction --publish
